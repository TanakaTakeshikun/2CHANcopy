# Features

簡易的な最低限の機能であり、JSを極力使用していないためPHPの勉強には最適になっている。

# Installation

[windows.php.net](https://windows.php.net/downloads/releases/)よりPHPをダウンロードし、phpという環境変数にダウンロードしたフォルダーのパスを通す

# Usage

```bash
git clone https://github.com/tenon-nonet/2CHANcopy
cd 2CHANcopy
```

MAMPより実行
